# AV360WEB #

Este repositório apresenta um projeto de Avaliação 360 focado no ensino superior, apresentando uma alternativa aberta para professores que possuem desejo de implementar essa metodologia.

Surgiu da necessidade de professores que buscavam integrar este método com a avaliação de trabalhos em grupo, mas encontravam barreiras devido aos custos de plataformas já existentes para meios empresariais.
